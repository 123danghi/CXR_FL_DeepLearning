About This Project

Breaking the Bias: Transforming Tuberculosis Detection with Deep Learning

Executive Summary

This project transformed an initial chest X-ray classification model with a critical flaw into a robust tuberculosis detection system with 99.43% sensitivity. Through fine-tuning and class balancing techniques, we addressed the model's original bias toward normal cases, dramatically improving TB detection from 3.43% to 99.43% while maintaining overall accuracy.

The Challenge: Hidden in Plain Sight

Our initial analysis revealed a deceptive scenario:

  • Overall accuracy: 83.55% (seemingly good)
  • Normal X-ray accuracy: 99.57% (excellent)
  • Tuberculosis accuracy: 3.43% (catastrophic)

This represented a dangerous bias—a model that would miss 97% of tuberculosis cases while appearing successful based on overall metrics.

Our Approach: Precision Rebalancing

We applied several strategic interventions:

  1. Architecture Refocusing: Converted from 3-class to binary classification
  2. Class Weighting: Applied 5x weight to TB cases in loss function
  3. Data Rebalancing: Oversampled TB images to match normal cases
  4. Training Optimization: Prioritized TB detection over overall accuracy
Results: From Failure to Excellence
Metric Original Model Improved Model
Overall Accuracy 83.55% 83.36%
Normal Detection 99.57% 80.14%
TB Detection 3.43% 99.43%

The confusion matrix tells the story:

  • Before: 676 missed TB cases out of 700
  • After: Only 4 missed TB cases out of 700
Clinical Impact

Our improved model makes the right trade-off for medical screening:

  • High Sensitivity: Captures 99.43% of TB cases
  • Appropriate Specificity: 80.14% accuracy on normal cases
  • Patient-Centered: Prioritizes catching disease over avoiding false positives
Key Insights
  1. Beware of Aggregate Metrics: Overall accuracy can hide critical failures
  2. Context Matters: Medical applications require different performance priorities
  3. Class Imbalance: Must be addressed directly in both data and training
  4. Intentional Bias: Sometimes needed to counteract inherent dataset bias
